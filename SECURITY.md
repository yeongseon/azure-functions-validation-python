# Security Policy

## Reporting a Vulnerability

We take security seriously. If you discover a security vulnerability, please report it responsibly.

### Preferred: GitHub Security Advisory

1. Go to the Security Advisories page:
   https://github.com/yeongseon/azure-functions-validation/security/advisories/new
2. Click "Report a vulnerability"
3. Submit details in the private advisory

### Alternative: Email

If you prefer email, contact: yeongseon.choe@gmail.com

### Response Timeline

- Initial response: within 48 hours
- Status update: within 7 days

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |
| < 0.1   | :x:                |
